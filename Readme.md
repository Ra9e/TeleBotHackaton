# Первоначальная настройка

Чтобы запустить бота, вам понадобится установить следующие библиотеки, командой `pip install`:

+ **pyTelegramBotAPI**

+ **googletrans==3.1.0a0**

+ **gTTS==2.2.2**

+ **SpeechRecognition**

+ **pyffmpeg**

### Не забудьте ввести ваш токен в файле `config.py`
